# VanillaSAP

Vanilla JS のSPAサンプル

## 開発環境
- node v14系

## 初回セットアップ
クローン後以下のコマンドでnodeモジュールをインストールする
```shell
npm install
```

## 開発方法

ローカルWEBサーバーを起動（起動後 http://localhost:8000 でアプリケーションを表示できる）
```shell
npm run serve
```

sassファイルの監視および自動コンパイル
```shell
npm run sass-watch
```

## ビルド方法

```shell
npm run webpack
```
